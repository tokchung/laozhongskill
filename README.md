## 已收录 Skill

### 看见自己 · `laozhong-kanjian`

> 从"我就是这样"，走到"原来是这样——那我可以那样。"

**适合谁：**
- 总在关系、生活里反复栽在同一个地方，却说不清为什么；
- 被贴过标签——"你太敏感""你想太多""你太作"——听多了自己也信了；
- 分手、受挫后心里冒出一句："我到底是个什么样的人？"
- 想改变、想成长，但觉得"了解自己"这事太玄，不知道从哪下手。

**你能拿到什么：** 6 层自我了解框架 + 三重辨析 + 松绑四层 + 防卡壳机制 + 完整实操案例。

**触发方式：** `/看见自己` 或说"看见自己""我不了解自己""为什么我总在同一个地方栽跟头"。


### 探索聊天室 · `laozhong-tansuo`

> 从"我被困住了"，走到"原来我还有这条路。"

三位心理学大师——弗洛伊德、阿德勒、荣格——以「创伤」「超越」「整合」的身份出场，按锁→推→拉递进结构，带你走一遍从看见到选择到整合的完整路径。全程不露名字，最后才揭晓。

**适合谁：**
- 心里堵着一件事，想找人聊聊，但不知道从哪说起；
- 试过"想开点""向前看""翻篇了"，但根本翻不过去；
- 在感情里反复遇到同一种人、在关系里反复掉进同一个模式；
- 不是来找标准答案的——是来找"我到底卡在哪一步"的；
- 对心理学好奇，但不想读教科书——想用对话的方式感受三种流派的碰撞。

**你能拿到什么：** 三人递进式深度对话 + 6 层追问 + 被耽误的事引导 + 一条从看见到行动的路径。不是给你建议，是让你看见自己卡在哪。

**适用场景：** 情感困扰（放不下前任/总遇渣男/反复分手）、社交困扰（讨好/不敢拒绝/不被喜欢）、家庭困扰（和父母处不好/在乎家人看法）、工作困扰（自我怀疑/不敢争取/反复栽坑）。

**触发方式：** `/探索聊天室` 或说"探索聊天室""我想聊聊我的困扰""我被困住了"。

## 两种使用方式

不管你会不会用 AI，都能用。

### 方式一：Skill 版（推荐，完整体验）★

安装后，AI 会化身对应 skill 的引导者，带着你一步一步走完全程。

```bash
npx skills add tokchung/laozhongskill
```

**前置条件：**
1. **Node.js** —— `npx` 命令来自它。没有的话去 [nodejs.org](https://nodejs.org) 下载安装（选 LTS 版）
2. **git** —— 工具靠它下载代码。没有的话去 [git-scm.com](https://git-scm.com) 安装
3. **至少一个受支持的 AI 工具** —— 支持 70+ 主流 AI 工具：Claude Code · Codex · Cursor · Windsurf · CodeBuddy · Zencoder · Qwen Code · Trae · Continue · Cline · Roo Code · Kilo Code · Goose · Droid · Junie · Lingma · iFlow CLI · Tabnine 等

**常见问题：**
- `npx: command not found` → 没装 Node.js，先装 Node.js
- `Cloning repository` 卡住 → 网络问题，确认能访问 github.com
- 装完在 AI 工具里没反应 → 重启一下 AI 工具，让它重新加载 skill

安装后在 AI 工具里输入对应触发词即可：
- `/看见自己` — 结构化了解自己
- `/探索聊天室` — 三位大师陪你拆解困扰

### 方式二：知识包（自己拿去用）

如果你不用上面那些 AI 工具，我们公开了完整的方法论文档。你可以：
- 当文档读，跟着走
- 贴到你自己的 AI 提示词里
- 拿去做 RAG、做自己的工具

所有文件在 `知识库/` 目录下，不依赖 skill 框架，独立可用。
