## 这是什么

"看见自己"是老钟（钟涛/芝心咨询）原创的自我了解框架。核心理念：**了解自己是知识问题，不是心理问题。我们以为的"缺点"，可能只是放错了位置的"特点"。**

你不需要"顿悟"，不需要被别人诊断。照着走，就能看见。

### 你能拿到什么

- **1 套 6 层自我了解框架**（素材 → 概念 → 溯源 → 转化 → 行动 → 快照），每一步都有具体的操作指引
- **5 个老钟原创核心概念**：松绑、松绑四层、三重辨析、概念松绑判断、"缺点是放错位置的特点"
- **1 份完整实操案例**（"敏感"走完全程，从"我想太多"到"我有洞察力天赋"）
- **1 套防卡壳机制**：每一步预埋了"走到这会卡住怎么办"的应对
- **13 种素材标签体系**：让你把关于自己的碎片信息，变成一个可查阅、可迭代的知识库

全部免费，不需要申请。

## 三种使用方式

不管你会不会用 AI，都能用。

### 方式一：Skill 版（Claude Code，完整体验）★ 推荐

在 Claude Code 中安装，AI 会化身"看见自己"的引导者，带着你一步一步走完全程。

```bash
npx skills add tokchung/laozhongskill
```

安装后在 Claude Code 里输入 `/看见自己` 或直接说"看见自己"，即可开始。

### 方式二：知识包（自己拿去用）

如果你不用 Claude Code，我们公开了完整的方法论文档。你可以：
- 当文档读，跟着走
- 贴到你自己的 AI 提示词里
- 拿去做 RAG、做自己的工具

所有文件在 `知识库/` 目录下，不依赖 skill 框架，独立可用。

### 方式三：公众号版（即将上线）

关注微信公众号「芝心咨询」，直接发消息就能用。开发中。
